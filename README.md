# Mines Game 

## Технологии

- **React 19** - UI 
- **Zustand** - управление состоянием 
- **Vite** - сборщик
- **Tailwind CSS** - стилизация
- **Lucide React** - иконки

## Что можно улучшить

1. **Интеграция с Backend API**
   - Аутентификация пользователей
   - Серверная валидация ставок
   - Хранение баланса на сервере
   - Генерация мин на сервере 

2. **Доказательство честновти**
   - Серверный сид для генерации поля
   - Клиентский сид
   - Возможность проверки результата

3. **Безопасность**
   - Rate limiting для API
   - Валидация всех входных данных
   - Защита от манипуляций с localStorage
